## 简介
2016.10 – 2016.11 物资申请管理系统，源于练手项目，基于nodejs开发的网站，使用MySQL做数据库，自己写的服务端路由，主要功能是查询物资申请表，填写物资申请表，审批物资申请表，查看物资库存，修改物资库存。
